# Fitness Club Lab (Python)

Консольная информационная система для фитнес‑клуба:
- учет клиентов
- учет абонементов
- расписание тренировок
- контроль посещения
- напоминания о продлении

## Запуск
Требования: Python 3.10+

Вариант 1 (как модуль):
```bash
python -m hospital_system
```

Вариант 2 (если ставите как пакет через pip):
```bash
pip install -e .
fitness-club
```

Данные хранятся в папке `data/` (JSON файлы).
